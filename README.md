## Установка зависимостей

Установка зависимостей:

```bash
# npm
npm install
```

## Запуск сервера разработки

Запуск сервера на `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Сборка

Сборка прод:

```bash
# npm
npm run build
```

Locally preview production build:

```bash
# npm
npm run preview
```
